# Wordle Solver

Guessing algrorithm for Wordle written in Python

## Installation

Either download and run the published exe or install the required modules and run guesser.py manually.

```bash
python3 guesser.py
```

## Usage

```python
import guesser

# Yields a word until word isn't correct

guesser.guess_word()
```
